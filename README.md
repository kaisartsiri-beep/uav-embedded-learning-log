# UAV & Embedded Systems Learning Log

This repository documents my learning progress in UAV systems, electronics, embedded systems, and drone platform integration.

The purpose of this repository is not to present professional UAV engineering experience, but to organize my beginner-level technical notes while I build the fundamentals needed to understand UAV architecture, payload integration, power systems, sensors, communication links, and embedded electronics.

## Purpose

This learning log helps me:

- Build a structured understanding of UAV systems
- Review basic electronics and embedded systems concepts
- Organize notes from academic work, self-study, documentation, and technical research
- Connect electrical and electronics engineering fundamentals with practical UAV applications
- Prepare for hands-on engineering opportunities involving UAV systems, electronics, testing, and technical documentation

## Current Learning Areas

- UAV system architecture
- Fixed-wing and VTOL UAV concepts
- Flight controllers
- GNSS and navigation sensors
- Telemetry and communication links
- IMU sensors
- ESCs and BLDC motors
- Payload integration
- Power systems and battery basics
- Bill of Materials basics
- Technical documentation
